Problem Archive

I've always been interested in trying out different puzzles and I've dabbled a bit in contests with some success. Throughout my problem-solving journey, I've noticed some interesting patterns and created some original problems. I used to post them on Brilliant.org, but the website changed its format and no longer has user-posted content. So I decided I'll put my original problems here.
